## moodleプラグイン
以下のような記述をすることでアスキーアートをMoodleのページに埋め込みます。


```
[aa_image]
　 　 　　　＿＿＿_ 
　 　　　／　　 　 　＼ 
　　　／　 _ノ 　ヽ､_　 ＼ 
　 ／ oﾟ(（●）) (（●）)ﾟo ＼ 　ほんとはVIPでやりたいんだお… 
　 |　　　　 （__人__）　　　　| 
　 ＼　　 　 ｀ ⌒´ 　 　 ／ 

[/aa_image]
```

このプラグインはsyntaxhighlighterを参考に作成しました。
https://moodle.org/plugins/filter_syntaxhighlighter
